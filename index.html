Bienvenue sur le site vitrine de l'application Mijotons

L'application qui allie écologie et économie !

Mijotons est une application Android gratuite que nous avons dans un premier temps, choisi de développer afin d'aider les étudiants de l'université à s'organiser plus facilement pour la gestion de leur repas.

Mijotons, l'application multifonctionnelle

En alliant l'écologie et l'économie, notre application vous permet de cuisiner à partir des ingrédients que vous avez dans vos placards. Mais ce n'est pas tout !

  • Affichage de recette en fonction des ingrédients sélectionnés
  • Mise en favoris de recettes
  • Consultation d'historique
  • Gestion de liste de courses
  • Recherche de restaurants à proximité

Technologies utilisées pour la production

Voici la liste exhaustive de l'ensemble des fonctionnalités que nous avons utilisées pour produire notre application.

Conception

Modelio

Logiciel qui nous a permis de faire toute la conception de notre application.

Planification

Gantt

Logiciel qui nous a permis d'organiser et dater les tâches de notre projet.

Prototypage

Balsamiq Wireframes

Logiciel qui nous a permis de réaliser les maquettes et prototypes de notre appli.

Versionning

Github & Git

Logiciel qui nous a permis de gérer le travail à plusieurs, le versionning et le stockage de notre projet.

Développement

Android Studio

IDE qui nous a permis de coder et tester notre application.

API

Npoint.io

Webservice qui nous a permis de produire facilement nos API à partir de JSON.
